# Bioinformática
Repositorio con los ejercicios, prácticas y proyectos realizados durante el máster en bioinformática y biología computacional de la universidad autónoma de madrid (UAM).

### Contenido del repositorio
* Herramientas de programación (ejercicios introductorios con comandos)
    * Python: Elaboración de Scripts
    * Linux: Elaboracion de script en Bash y, automatización y prácticas de terminal.
    * Bases de datos: elaboracion de modelado relacional y consultas SQL.

* Procesado y manejo de datos masivos
    * Linux: Elaboración de comandos de shell  para el procesado y manejo de datos
    * Bases de datos: Elaboración de consultas en bases de datos NO-SQL
    * Biopython: Elaboracion de programas para el manejo y procasamiento de datos estructurados con la utilización de APIs biomédicas

* Fundamentos de secuenciacion de alto rendimiento
* Analisis de secuencias 
* Filogenia molecular
* Aprendizaje automático
* Algoritmos en bioinformática
* Bioinformática estructural
* Transcriptómica, regulación genómica y epigenómica
* Redes biólogicas 
* Metagenómica

Nota: Todo el código es estrictamente académico y se publica con fines educativos.
